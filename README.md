# 🌐 Personal Portfolio

My personal developer portfolio — built with **JavaScript** and **Tailwind CSS** to showcase my projects, skills, and experience.  
It’s designed to be clean, responsive, and fast, with smooth animations and a modern UI.

---

## 🚀 Features

- Fully responsive design  
- Built with JavaScript + Tailwind CSS  
- Smooth transitions and modern layout  
- Project showcase section  
- Contact links (Email, GitHub, LinkedIn, etc.)  
- Resume download button  

---

## 🛠️ Tech Stack

- **Frontend:** React, Tailwind CSS  
- **Deployment:** Vercel  
- **Version Control:** Git & GitHub
